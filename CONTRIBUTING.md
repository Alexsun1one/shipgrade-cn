# Contributing

ShipGrade CN 接受能提高真实交付质量的贡献,不接受只增加口号或提示词堆叠的贡献。

## Good Contributions

- 新增可验证模板、doctor check、eval case、真实脱敏 transcript。
- 补充公开、许可证清晰、高信号来源的结构证据。
- 改进中文小白入口,同时不降低专业安全边界。
- 修复国内网络、Windows/Mac 混合、多 agent 协作中的真实坑。

## Required Evidence

每个 PR/变更至少说明:

- 交付了什么。
- 用什么命令或文档验证。
- 来源和许可证边界。
- 是否触及 secret/token/cookie/session/auth/private key。
- 下一位维护者从哪里接手。

## Rejected Contributions

- 未授权私有仓库、泄漏源码、系统提示词归档。
- 明文 secret、token、cookie、session、auth database、private key。
- 许可证不清的正文搬运。
- 只加“更强”“世界级”等形容词,没有验证或 artifact。
