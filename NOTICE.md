# Notice

ShipGrade CN is an original Chinese engineering skill package distilled from public, license-reviewed engineering sources.

## Provenance

- Full source map: `docs/source-attribution.md`
- Influence map: `docs/influence-map.md`
- Source-depth evidence: `docs/source-depth-dossier.md`, `docs/deep-code-case-studies.md`
- Runtime and transcript evidence: `docs/runtime-execution-evidence.md`, `docs/transcript-evidence.md`

## License Handling

- Green-license sources are used for structural and workflow distillation.
- Attribution-review sources require downstream attribution.
- Share-alike sources are treated as review-only unless compatible attribution/share-alike handling is explicitly preserved.
- Missing or unclear license sources are metadata-only.

## Safety Boundary

This package excludes private repositories, leaked source, system-prompt archives, secrets, browser profiles, cookies, auth/session databases, private keys, and API tokens.
